# cperceptron

Projeto educacional em C para estudo de **Multi-Layer Perceptrons (MLP)**.  
Implementa, do zero, feedforward e backpropagation, com foco em entender os fundamentos de redes neurais, controle de memória e desempenho, sem uso de bibliotecas externas de ML.

- Linguagem: C (C99/C11)
- Objetivo: aprendizado e experimentação
- Status: em desenvolvimento
